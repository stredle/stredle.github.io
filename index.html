Stredle Cryptocurrency

Stredle's Features

Fast transactions

Stredle transactions are fast, it takes less than a minute for a transaction to be confirmed.

Secure network

Stredle's network is constantly updated to guarantee maximum security.

Easy-to-use Wallet

Since Stredle is a fork of Litecoin, it's wallet is nearly the same, and the Litecoin Wallet is very user-friendly, and so is Stredle's

Get Started With Stredle

If you are new to cryptocurrencies, you may need a little help to know how to properly use them.

Create Your Wallet

The first thing to do when starting using stredle is to create a wallet to store your STEs. Normally, a wallet is automatically generated when the Stredle wallet is first opened

Buy or Mine Stredle

Having a wallet to store Stredles is good, but having Stredles is better, to achieve such a thing, you may want to mine Stredles or to buy them.

OK !

Now that you're all set up, you may want to use Stredle as a normal currency, or, at least, as a market share
